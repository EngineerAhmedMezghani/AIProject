# Projet d'Intelligence Artificielle

## Description
Ce projet est une introduction à l'intelligence artificielle, abordant les concepts fondamentaux et les applications pratiques.

## Installation
1. Clonez le dépôt : `git clone <url-du-depot>`
2. Accédez au répertoire du projet : `cd Projet`
3. Installez les dépendances : `pip install -r requirements.txt`

## Utilisation
Pour exécuter le programme principal, utilisez la commande suivante :
```bash
python main.py
```
